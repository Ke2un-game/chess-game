# ♟️ Chess Game (Python & Pygame)

A clean, responsive 2D Chess game built with Python and Pygame. It features multiple AI difficulty levels, a 2-player local pass-and-play mode, move highlighting, and a full undo system.

---

## ✨ Features

* 🤖 **Bot AI Mode:** Play against 4 different engine difficulty stages:
  * Candidate Master
  * FIDE Master
  * International Master
  * Grandmaster
* 👥 **2-Player Mode:** Pass and play locally with a friend on the same device.
* ↩️ **Undo Move:** Easily undo accidental moves in both Bot and 2-Player modes.
* 🎯 **Visual Highlighting:** 
  * Selected piece highlight
  * Valid move indicator dots
  * Last move yellow highlight
  * Red king highlight when in Check
* 📱 **Portrait UI:** Optimized layout tailored for mobile and portrait screens (Pydroid 3 friendly).

---

## 🛠️ Tech Stack & Requirements

* **Language:** Python 3
* **Libraries:**
  * `pygame`
  * `chess` (python-chess)
